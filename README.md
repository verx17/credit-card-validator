# Credit Card Validator

## 📋 Описание проекта

Консольная утилита на языке Go для быстрой валидации номеров банковских карт и определения банка-эмитента.

Приложение выполняет три этапа проверки:
1. Длина номера (13-19 символов), допустимые символы (только цифры).
2. Проверка корректности номера по [**алгоритму Луна**](https://ru.wikipedia.org/wiki/%D0%90%D0%BB%D0%B3%D0%BE%D1%80%D0%B8%D1%82%D0%BC_%D0%9B%D1%83%D0%BD%D0%B0).
3. Определение банка по первым 6 цифрам карты (BIN) на основе базы данных.

## 🛠 Основные функции

- **Загрузка данных из CSV:** База банков подгружается из внешнего файла `banks.txt` при старте.
- **Интерактивный ввод:** Бесконечный цикл обработки пользовательских запросов.
- **Очистка данных:** Автоматически удаляет пробелы, дефисы из введенного номера (например, ввод `4444 3333...` будет обработан корректно).
- **Валидация формата:** Строгая проверка на наличие посторонних символов и соответствие длине стандарта (13-19 цифр).
- **Алгоритм Луна:** Реализация стандартной проверки контрольной суммы для защиты от опечаток.
- **Поиск по диапазонам BIN:** Определение банка путем проверки вхождения BIN в заданные диапазоны.

## 🚀 Установка и запуск

### Предварительные требования
- Установленный **Go** (версии 1.22 или выше).
### Установка

1. **Клонируйте репозиторий**
   ```bash
   git clone https://github.com/verx17/credit-card-validator.git
   ```
2. **Перейдите в директорию проекта**
   ```bash
   cd credit-card-validator
   ```
3. **Подготовка файлов**
Создайте файл данных `banks.txt` в одной директории с программой. Формат файла: `Название Банка,BIN_Начало,BIN_Конец`

**Пример `banks.txt`:**
```
Lunar Bank,400000,499999
Mars Credit Union,500000,599999
Venus Express Bank,600000,699999
```
### Запуск программы
Откройте терминал в папке с проектом и выполните команду:
```bash
go run main.go
```

Или скомпилируйте бинарный файл:
```bash
go build -o Validator
./Validator
```

## Примеры использования

**Сценарий 1: Успешная валидация**

Пользователь вводит корректный номер карты, который есть в базе (в данном примере BIN 555544 попадает в Mars Credit Union).
```Plaintext
Загрузка программы валидации банковских карт космической системы...
Введите номер карты:
> 5555 4444 3333 2226
Карта 5555444433332226 относится к банку Mars Credit Union
```

**Сценарий 2: Ошибка алгоритма Луна**

Пользователь ошибся в оной цифре (контрольная сумма не совпала)
```Plaintext
Введите номер карты:
> 4000001234567890
Введённый номер карты не прошёл проверку
```

**Сценарий 3: Банк не найден**

Номер карты валиден по алгоритму Луна, но его BIN отсутствует в файле banks.txt.
```Plaintext
Введите номер карты:
> 1111222233334444
Ошибка: не удалось определить Банк карты 1111222233334444
```

**Сценарий 4: Выход из программы**

Нажатие `<Ctrl c>` завершает работу.
```Plaintext
Введите номер карты:
> 
Закрытие программы...
```
